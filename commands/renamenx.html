加入版本 。

时间复杂度： O(1)。

当且仅当 newkey 不存在时，将 key 改名为 newkey 。

当 key 不存在时，返回一个错误。

Return value

Integer reply, specifically:

  • 修改成功时，返回 1 。
  • 如果 newkey 已经存在，返回 0 。

Examples

redis>  SET mykey "Hello" 
OK
redis>  SET myotherkey "World" 
OK
redis>  RENAMENX mykey myotherkey 
(integer) 0
redis>  GET myotherkey 
"World"
redis>
